FAQ

เกียรติบัตรใส่พอร์ต TCAS รอบ 1 ได้จริงไหม? +
ได้จริง — MicroCredential จาก KMUTT เป็น verifiable digital badge ที่มี metadata ครบ (สมรรถนะที่ผ่าน, ผู้ประเมิน, หลักฐาน) กรรมการกดลิงก์ดูได้ทันที
ต้องมีพื้นฐานอะไรบ้าง? +
ไม่จำเป็นต้องมีพื้นฐานด้านแพทย์ — ค่ายออกแบบมาสำหรับน้องที่ "สนใจ" ไม่ใช่คนที่ "พร้อมแล้ว" ทุก session มี facilitator ดูแล
ค่าใช้จ่ายรวมอะไรบ้าง? +
รวมทุกอย่าง: ที่พัก 2 คืน อาหารทุกมื้อ วัสดุ workshop ทั้งหมด MicroCredential KMUTT, SET Certificate, AI Certificate และ Portfolio Package ไม่มีค่าใช้จ่ายเพิ่มเติม
ต่างจากค่ายอยากเป็นหมอ/MEDCHIC อย่างไร? +
ค่ายอยากเป็นหมอ (จุฬา) เน้นสัมผัสชีวิตหมอ แต่แข่งขันสูง MEDCHIC เน้น hackathon ส่วน Path to Med เน้น "career clarity" — น้องที่ยังไม่แน่ใจสมัครได้เลย
สมัครคู่กับเพื่อนได้ไหม? +
ได้เลย! สมัครคู่กับเพื่อน ลดเพิ่มอีก 300 บาท/คน ระบุชื่อเพื่อนในฟอร์มสมัคร
